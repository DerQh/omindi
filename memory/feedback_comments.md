Always add a short comment above every function explaining what it does.

**Why:** User explicitly requested this — they want functions to be self-documenting via comments, not just by name.

**How to apply:** Add a one-line `//` comment directly above every function definition (hooks, handlers, helpers, components). Keep it concise — one sentence describing the purpose, not the implementation.
